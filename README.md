#  Application de Gestion Bancaire en Java

##  Présentation
Cette application est un système de gestion bancaire développé en **Java** en utilisant les principes de la **programmation orientée objet (POO)**.  
Elle permet de gérer des clients, différents types de comptes bancaires et les opérations associées (dépôts, retraits, virements, intérêts).

Le projet met en œuvre plusieurs concepts avancés tels que :
- l’héritage
- le polymorphisme
- les exceptions personnalisées
- le pattern Factory
- les interfaces et stratégies
- la modélisation UML

---

##  Objectifs du projet
- Modéliser un système bancaire réel
- Appliquer les bonnes pratiques de conception orientée objet
- Gérer les erreurs métier via des exceptions personnalisées
- Séparer clairement les responsabilités (modèle, service, factory, etc.)

---

##  Fonctionnalités principales
- Création de clients
- Gestion de plusieurs types de comptes :
    - Compte Courant
    - Compte Épargne
    - Compte Entreprise
- Dépôt et retrait d’argent
- Virements entre comptes
- Application des intérêts pour les comptes épargne
- Historique des transactions
- Gestion des erreurs (solde insuffisant, montant invalide, limites dépassées)

---

##  Technologies utilisées
- **Java**
- Programmation Orientée Objet (POO)
- UML (PlantUML)
- Exceptions personnalisées
- JavaDoc

---
